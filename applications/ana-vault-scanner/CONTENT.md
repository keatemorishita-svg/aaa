# Ana — Obsidian Vault Scanner

> Theme clustering, heatmaps, dormant notes, AI cognitive insights. Zero config.

**Repo**: [keatemorishita-svg/ana](https://github.com/keatemorishita-svg/ana)

---

## What it does

Ana scans an Obsidian knowledge base to analyze notes, detect patterns, and generate cognitive summaries. It outputs findings to `_Ana/WeeklyManifesto.md`.

## Key features

- **Theme Clustering**: Auto-groups notes by tags + folders
- **Activity Heatmap**: 90-day editing history visualization
- **Unexpected Connections**: Random pairing, shared links, temporal co-occurrence, shared tags
- **Dormancy Detection**: Flags notes unmodified for 90 days
- **Guiding Questions**: Reflection prompts based on vault state
- **Dream Log**: Lightweight daily scan during idle time

## AAA Mapping

Ana is a **Cognition Layer** application — it helps you understand your own knowledge base. It's meta-cognition for your second brain.
