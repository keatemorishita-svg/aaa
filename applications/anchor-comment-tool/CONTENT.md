# Anchor · 评论区神器

> Every comment is an anchor cast out. AI-powered comment generation for genuine connection.

**Repo**: [keatemorishita-svg/anchor](https://github.com/keatemorishita-svg/anchor)

---

## What it does

Anchor generates high-quality comments for Chinese short-video platforms. Paste a creator's content → select intent and style → AI generates a comment designed to foster genuine connection.

## Key features

- **3 Intent Modes**: Networking (建联), Self-Warming (暖场), Alt-Account (小号)
- **6 Styles**: Resonance, Supplement, Question, Lighthearted, Professional, Restrained
- **4 Tones**: Humble, Casual, Playful, Gentle
- **Privacy-Focused**: DeepSeek API key stays local; no third-party data collection
- **Tech Stack**: Flask + DeepSeek Chat API + Vanilla JS/CSS (zero build step)

## AAA Mapping

Anchor is an **Execution Layer** application — it performs a specific action (comment generation) with customizable parameters. The intent/style/tone matrix is a mini protocol for social interaction.
