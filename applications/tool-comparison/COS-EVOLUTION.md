# COS-EVOLUTION

## Current State
- **Maturity**: [x] Stable
- **Known weaknesses**: 评分带有主观性（基于用户实际使用而非标准化测试）；工具版本迭代快，积分表需要持续更新；7 款工具的覆盖范围偏向西方市场

## Evolution Invitation
- **Best contribution type**: PR — 更新工具评分、增加新工具
- **What needs improvement most**: 将主观评分转化为可复现的评分标准（每个维度的评分 rubric）
- **What should NOT be changed**: Agent-first 视角（这是本框架的核心区分点，改为补全-first 相当于不同的项目，请 Fork）

## Fork Policy
- **Fork friendly?**: [x] Yes, fork anything — 特别欢迎 Fork 为其他领域的工具对比框架
- **Merge preference**: 工具评分更新欢迎 Merge 回上游；框架性改动请保持为独立 Fork
- **Succession**: 如果原维护者停止更新，任何人可以 Fork 并继续维护积分表。工具对比的价值在于时效性。
