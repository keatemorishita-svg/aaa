## How to add your application

1. Fork this repository
2. Create `applications/<your-app-name>/` with 4 files:
   - `CONTENT.md` — your content
   - `COS-META.md` — architecture mapping ([template](../APP-STORE-PROTOCOL.md#-cos-metamd--架构映射))
   - `COS-SOP.md` — method trace ([template](../APP-STORE-PROTOCOL.md#-cos-sopmd--方法溯源))
   - `COS-EVOLUTION.md` — evolution invitation ([template](../APP-STORE-PROTOCOL.md#-cos-evolutionmd--进化邀请))
3. Add your application to this INDEX.md
4. Open a PR

→ Full protocol: [APP-STORE-PROTOCOL.md](../APP-STORE-PROTOCOL.md)
