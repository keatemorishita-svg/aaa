# Brand — Typography

## Primary Typefaces

### English
- **Display / Headlines**: Inter (sans-serif, geometric, modern)
- **Body / Long Text**: SF Pro Text or Inter
- **Code / Technical**: JetBrains Mono

### Chinese / CJK
- **Display**: 思源黑体 (Source Han Sans) / HarmonyOS Sans
- **Body**: 思源宋体 (Source Han Serif) for long text

---

## Type Scale

| Level | Size | Weight | Use |
|---|---|---|---|
| H1 | 48px | Bold (700) | Page titles |
| H2 | 32px | SemiBold (600) | Section headers |
| H3 | 24px | Medium (500) | Subsection headers |
| H4 | 18px | Medium (500) | Card titles |
| Body | 16px | Regular (400) | Paragraphs |
| Small | 14px | Regular (400) | Metadata, captions |
| Code | 14px | Regular (400) | Code blocks |

---

## Principles

- **Clarity over decoration** — the ideas are complex enough; the type should be simple
- **High contrast** — dark backgrounds, light text
- **Space is part of the design** — generous line height (1.6–1.8), ample margins
- **Monospace for structure** — code blocks, diagrams, and protocol definitions use monospace
