# Brand — Color System

## Primary Palette

| Color | Hex | Role |
|---|---|---|
| Deep Black | `#0B0F19` | Foundational structure, space, depth |
| Evolution Blue | `#4DA3FF` | Cognition, AI, protocol, clarity |
| Emergence Green | `#2EE59D` | Evolution, growth, new structure |

---

## Extended Palette

| Color | Hex | Role |
|---|---|---|
| Void Black | `#060910` | Deepest background layers |
| Surface Dark | `#141929` | Cards, panels, UI surfaces |
| Text Primary | `#E8ECF1` | Primary text on dark |
| Text Secondary | `#8B95A8` | Secondary text, metadata |
| Accent Amber | `#FFB347` | Warnings, selection, attention |
| Accent Red | `#FF6B6B` | Errors, destructive actions |

---

## Usage Principles

- **Dark-first** — the system lives in deep structure; light is the inversion
- **Blue for cognition** — anything related to understanding, modeling, or AI
- **Green for evolution** — anything related to change, growth, or adaptation
- **Gradient transitions** — blue → green gradients represent the cognition → evolution pipeline
