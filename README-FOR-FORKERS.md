# For Forkers

If you are reading this, you are not a user.

You are a potential branch of evolution.

---

## What you can do

- Change the definition
- Rebuild the system
- Extend the model
- Disagree completely
- Replace the primitives
- Add new layers
- Remove what doesn't work

---

## Rule

Fork is not copying.

Fork is evolution.

---

## How this project dies

This project dies if nobody forks it.

It lives if people take it in directions the original never imagined.

---

## What we hope you do

Don't try to be faithful to the original.

Be useful.
