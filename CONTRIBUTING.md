# Contributing to AAA

AAA evolves through participation.

You can contribute by:

- proposing new models
- forking interpretations
- extending frameworks
- challenging assumptions

---

## Rule of this project

There is no single authoritative version.

Every contribution is a potential branch of evolution.

---

## How to contribute

1. Fork this repository
2. Make your changes — any changes
3. Open a pull request describing your interpretation

There is no wrong contribution. Only divergent evolution.
