# AAA — Discussion

## How AAA evolves

This project does not have a roadmap.

It has a discussion — an open space where interpretations diverge and converge.

---

## Discussion Principles

- **Divergence is not failure.** It is the mechanism of evolution.
- **There is no "correct" interpretation.** There are only more useful and less useful ones.
- **Challenge the core.** The primitives, the loop, the architecture — all are open to reinterpretation.
- **Build bridges.** Connect AAA to your own domain and share what you find.

---

## How to start a new thread

Open a GitHub Discussion with one of these labels:

- `challenge` — questioning a core concept
- `extension` — proposing a new layer or model
- `application` — applying AAA to a real system
- `meta` — discussing how the project itself evolves
