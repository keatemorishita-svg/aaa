# Language Note: 演化 vs 进化

> A terminology decision that is also a philosophical stance.

---

## The Distinction

In Chinese, there are two words for "evolution":

| Term | Connotation | Time Scale | Direction |
|---|---|---|---|
| **演化** (yǎnhuà) | Slow, gradual, undirected change | Geological / Darwinian | None — it just happens |
| **进化** (jìnhuà) | Progressive, accelerated, directional improvement | Compressed / engineered | Forward — it gets better |

English uses one word ("evolution") for both. This collapses an important distinction.

---

## Why AAA Uses 进化

AAA makes a specific claim:

> AI changes the speed and scale of the evolution loop.
> What once took millennia now takes days.
> What once required a species now requires a repository.

Using 演化 to describe this would be a self-betrayal. The slow, undirected connotation of 演化 contradicts the core thesis: that AI-accelerated evolution is qualitatively different from natural selection.

**进化 is the word. 共同进化 is the stance.**

---

## Where the distinction matters

| Context | Use | Reason |
|---|---|---|
| Describing AAA mechanisms | 进化 | AI-accelerated, directional |
| Describing the project's self-evolution | 进化 | Intentional, fast, designed to improve |
| Describing Darwinian natural selection | 演化 | Slow, undirected, no designer |
| Describing historical system change (pre-AI) | 演化 | The old pace |
| Describing GitHub ecosystem dynamics | 进化 | Fork → Merge at internet speed |

---

## In English

English lacks the 演化/进化 distinction. The workaround:

- Use "evolve" as the baseline
- When emphasis is needed: "accelerated evolution" / "rapid evolution" for 进化
- When describing natural systems: "gradual evolution" / "Darwinian evolution" for 演化

---

## The core quote — bilingual

> A civilization begins with a theory. But it is more — a kernel, a method, a set of working applications, a memory, and an evolution protocol — all inhabiting one space, evolving together.
>
> 一个文明首先是一套理论。但它不止于理论——它是一颗内核、一套方法、一组工作应用、一份记忆、一个进化协议——共处于一个空间，共同进化。

The English "evolving together" maps to the Chinese "共同进化" — not "共同演化". This is deliberate. In the AI era, evolution has a gear it never had before.
